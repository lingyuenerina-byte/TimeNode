# 刻度 (TimeNode)

刻度 (TimeNode) 是一个专为视频创作者打造的桌面端时码与进度条渲染引擎。它可以根据用户设置的时间节点，生成带有平滑进度条、章节标题和滚动时间码的视频素材，方便在后期剪辑软件（如 Premiere Pro、DaVinci Resolve）中作为叠加层使用。

## 功能特性

* **高精度平滑渲染**：底层采用 WebGL 亚像素抗锯齿（Sub-pixel Anti-aliasing）算法计算进度，彻底消除进度条在低速率移动时的像素级抖动与视觉卡顿。
* **多格式视频引擎**：支持输出标准的 H.264 绿幕视频 (MP4)，以及带有完整透明通道的 ProRes 4444 视频 (MOV)，完美契合专业后期工作流。
* **高度可定制化外观**：提供图形化界面调节轨道尺寸、颜色、描边、文字排版等参数，并支持挂载自定义的二次元游标图（Mascot）。
* **独立的本地沙盒管理**：
  * **免安装字体挂载**：软件会自动读取根目录 `fonts` 文件夹中的字体文件 (TTF/OTF/WOFF) 并直接渲染，无需繁琐的系统全局安装。
  * **预设系统**：所有的样式配置均可作为 JSON 预设文件保存在本地 `presets` 文件夹中，支持跨设备备份与快速导入。

## 安装与运行指南

本软件为绿色免安装的独立运行程序，无需配置任何代码环境。

1. 下载打包好的 `TimeNode.exe` 可执行文件。
2. **⚠️ 核心依赖**：本软件的视频编码底层依赖于 FFmpeg。请务必自行下载 `ffmpeg.exe`，并将其直接放置在与 `TimeNode.exe` 同级的目录中（或者将其添加到系统环境变量 PATH 中）。
3. 双击运行即可使用。

## 目录结构说明

软件首次运行后，会在可执行文件的同级目录下自动生成以下三个文件夹，请确保软件具有该目录的读写权限：
* `/fonts`：将你喜欢的字体文件拖入此处，即可在软件内直接调用。
* `/presets`：用于存放以 `.json` 格式保存的视觉预设配置。
* `/output`：默认的视频导出保存目录。
