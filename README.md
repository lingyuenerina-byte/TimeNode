# 刻度 (TimeNode)

刻度 (TimeNode) 是一个桌面端时码与进度条渲染工具。它可以根据用户设置的时间节点，生成带有平滑进度条、章节标题和滚动时间码的视频素材，方便在后期剪辑软件（如 Premiere Pro、DaVinci Resolve）中作为叠加层使用。

*注：本项目的核心架构设计与部分源代码借助 Google Gemini 生成。*

## 功能特性

* **高精度平滑渲染**：底层采用 WebGL 亚像素抗锯齿（Sub-pixel Anti-aliasing）算法计算进度，消除进度条在低速率移动时的视觉卡顿。
* **多格式视频输出**：支持输出标准的 H.264 绿幕视频 (MP4)，以及带有完整透明通道的 ProRes 4444 视频 (MOV)。
* **可定制化外观**：提供图形化界面调节轨道尺寸、颜色、描边、文字排版等参数，并支持挂载自定义的游标图。
* **独立的本地文件管理**：
  * **本地字体加载**：自动读取根目录 `fonts` 文件夹中的字体文件 (TTF/OTF/WOFF) 并直接渲染，无需进行系统级安装。
  * **预设系统**：样式配置可作为 JSON 预设文件保存在本地 `presets` 文件夹中，支持跨设备备份与导入。

## 安装与使用

本软件为免安装的独立运行程序。

1. 下载 `TimeNode.exe` 可执行文件。
2. **核心依赖**：本软件的视频编码依赖 FFmpeg。请自行下载 `ffmpeg.exe`，并将其直接放置在与 `TimeNode.exe` 同级的目录中（或添加至系统环境变量）。
3. 双击运行即可。

## 目录结构

软件首次运行后，会在当前目录下自动生成以下文件夹：
* `/fonts`：用于存放用户导入的自定义字体文件。
* `/presets`：用于存放以 `.json` 格式保存的视觉预设配置。
* `/output`：默认的视频导出保存目录。
