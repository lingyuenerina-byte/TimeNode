刻度 (TimeNode) 是一个桌面端时码与进度条渲染工具。它可以根据用户设置的时间节点，生成带有平滑进度条、章节标题和滚动时间码的视频素材，方便在后期剪辑软件（如 Premiere Pro、DaVinci Resolve）中作为叠加层使用。

>**平台支持**：本项目目前仅支持 **Windows** 操作系统。

*注：本项目的核心架构设计与部分源代码借助 Google Gemini 生成。*

## 功能特性

* **高精度平滑渲染**：底层采用 WebGL 亚像素抗锯齿（Sub-pixel Anti-aliasing）算法计算进度，消除进度条在低速率移动时的视觉卡顿。
* **内置渲染引擎**：安装包已集成 FFmpeg，实现开箱即用，无需手动配置环境变量。
* **多格式视频输出**：支持输出标准的 H.264 绿幕视频 (MP4)，以及带有完整透明通道的 ProRes 4444 视频 (MOV)。
* **可定制化外观**：提供图形化界面调节轨道尺寸、颜色、描边、文字排版等参数，并支持挂载自定义的游标图。

## 安装与使用

### 方法一：直接下载（推荐）
请前往 [Releases 页面](../../releases) 下载最新的 `setup.exe` 安装包。该版本已包含所有运行依赖，安装后即可直接使用。

### 方法二：从源码构建（面向开发者）
如果你想自行编译：
1. 克隆本项目，执行 `npm install`。
2. **放置依赖**：本项目的构建脚本会自动寻找资源。请务必将 `ffmpeg.exe` 放置在 **`src-tauri/binaries/`** 目录下。
3. 调试：`npm run tauri dev`
4. 打包：`npm run tauri build`

## 目录结构
软件运行后会生成以下文件夹：
* `/fonts`：存放用户导入的自定义字体。
* `/presets`：存放 `.json` 格式的视觉预设。
* `/output`：默认视频导出目录。
